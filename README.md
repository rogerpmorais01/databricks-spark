### 🛠️ Como Executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/databricks-portfolio.git
   ```
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
3. Importe os notebooks para o Databricks via **File > Import**.
